# CSV Demo Analyser

Basic command-line CSV analysis tool.

This is a lightweight demonstration module showcasing structured data inspection
for portfolio and capability presentation purposes.

## Scope

- Works only with CSV files
- No external dependencies
- Pure Python (standard library only)

## Features

- Total row count
- Column listing
- Missing value count per column
- Basic data type inference (Integer / Float / Text)

## Usage

Run the script using:

python demo_analyser.py

You will be prompted to enter the CSV file name.

## Design Philosophy

- Minimal
- Environment-independent
- Dependency-free
- Suitable for automation pipelines

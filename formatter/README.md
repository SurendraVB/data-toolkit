# CSV to JSON Formatter (Demo Version)

Minimal command-line utility for converting CSV files to JSON format.

This is a simplified demonstration module provided for portfolio and capability presentation purposes.

## Scope

- Supports CSV → JSON conversion only
- No interactive interface
- No schema inference
- No advanced normalization
- Pure Python (standard library only)
- No external dependencies

## Features

- Reads CSV file using structured column headers
- Converts rows into JSON objects
- Writes formatted JSON output (indented)
- Basic error handling

## Usage

Run from terminal:

python3 demo_formatter.py input.csv output.json

Example:

python3 demo_formatter.py sample.csv sample.json

## Design Principles

- Minimal implementation
- Deterministic behavior
- Dependency-free execution
- Suitable for automation workflows

# CSV Duplicate Cleaner (Demo Version)

Lightweight command-line utility for removing exact duplicate rows from CSV files.

This is a simplified demonstration module showcasing structured data cleaning
capabilities for portfolio presentation purposes.

## Scope

- Works only with CSV files
- Removes exact duplicate rows
- Preserves header row
- Pure Python (standard library only)
- No external dependencies

## Features

- Reads input CSV file
- Detects exact duplicate rows
- Generates cleaned output file
- Displays summary statistics:
  - Original row count
  - Number of duplicates removed
  - Remaining rows

## Usage

Run the script:

python demo_cleaner.py

You will be prompted to provide:
- Input CSV file name
- Output CSV file name

## Design Principles

- Minimal implementation
- Dependency-free
- CLI-friendly
- Suitable for automation workflows
